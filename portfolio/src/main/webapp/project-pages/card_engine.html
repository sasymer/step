Card Game Engine

Java, JavaFX, data-driven design, inheritance, OOP

I worked in a team of 4 to create a card game engine. The idea was to create a data-driven design, allowing users to create custom card games through modifying only XML and property files. I worked on the back-end, implementing an inheritance hierarchy for possible actions that take place in a card game (transfer card between users, select the top card from the center deck, etc.).
Ultimately, my team created the capacity for relatively simple multiplayer games such as Old Maid, Go Fish, War, Slapjack, etc.
