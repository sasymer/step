Cell Simulation

Java, JavaFX

In a team of 3, I worked on the back-end to create a program that has the ability to run any kind of grid-based simulation, including predator-prey, fire spreading, rock paper scissors, game of life, etc. Using an inheritance hierarchy, I implemented all of the simulation rules. I also created functionality in the back-end to allow for various types of grid edges, such as a wrap-around grid. Additionally, simulations can be run on any size of cell neighborhood.
